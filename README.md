# AirSign Taser – File Transfer Using Hand Gestures

## 📌 Project Overview

AirSign Taser is a web-based prototype that allows users to transfer files using hand gestures detected through the camera.

## 🎯 Objective

To create a touchless file transfer system using gesture recognition.

## ⚙️ How It Works

1. The system accesses the user’s camera.
2. Hand gestures are detected using MediaPipe.
3. Different gestures trigger actions:

   * Open Hand → Send File
   * Closed Hand → Receive File
4. The selected file is processed and transferred (prototype/demo level).

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* MediaPipe (for hand tracking)

## ▶️ How to Run

1. Open the project link:
   https://yourusername.github.io/AirSign-Taser/
2. Allow camera access
3. Select a file
4. Perform gestures to trigger actions

## 🚀 Features

* Real-time hand gesture detection
* Camera-based interaction
* Simple UI for file handling
